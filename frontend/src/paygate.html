How it works

Three steps from zero to fully orchestrated payments.

01

Install PayGate

Connect via REST API or drop in our SDK. 5 lines of code to go live.

// Install the SDK
npm install @paygate/sdk

// Initialize and charge
import { PayGate } from '@paygate/sdk';

const pg = new PayGate('pk_live_...');

const charge = await pg.charges.create({
  amount: 499900,
  currency: 'INR',
  provider: 'auto'
});
02

Connect Your Providers

Add Stripe, Razorpay, PayU — any combination. Set routing rules per currency, region, or transaction value.
